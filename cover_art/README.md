# 🎨 Album Cover Images Setup

## 📁 How to Add Album Covers

Place your Taylor Swift album cover images in this folder!

### File Naming Convention:

The images should be named using the album name in **lowercase with underscores**:

**Examples:**
- `taylor_swift.jpg` - Taylor Swift (debut)
- `fearless.jpg` - Fearless
- `fearless_taylors_version.jpg` - Fearless (Taylor's Version)
- `speak_now.jpg` - Speak Now
- `red.jpg` - Red
- `red_taylors_version.jpg` - Red (Taylor's Version)
- `1989.jpg` - 1989
- `1989_taylors_version.jpg` - 1989 (Taylor's Version)
- `reputation.jpg` - Reputation
- `lover.jpg` - Lover
- `folklore.jpg` - Folklore
- `evermore.jpg` - Evermore
- `midnights.jpg` - Midnights

### Supported Formats:
- `.jpg`
- `.jpeg`
- `.png`
- `.webp`

### Recommended Size:
- **At least 300x300 pixels** (square)
- Higher resolution is better for quality!

### Where to Find Album Covers:
1. **Spotify** - Right-click on album → Copy Image
2. **Apple Music** - Screenshot album art
3. **Google Images** - Search "Taylor Swift [album name] cover"
4. **Amazon Music** - High quality album artwork

## 🚀 Quick Setup:

1. Save album cover images to this folder
2. Name them according to convention above
3. Run the app: `python app.py`
4. Watch beautiful floating album covers appear! ✨

**Note:** If an image is not found, the app will use a colored fallback based on the album's aesthetic.
