# Fonts

All fonts are self-hosted (no Google Fonts dependency for these).

| Font | File | License | Currently used for |
|---|---|---|---|
| Beauty Gadish | `BeautyGadish.otf` | Personal use | "Dear Diary," label |
| Betris and Daniel | `BetrisAndDaniel.otf`, `BetrisAndDanielItalic.otf` | Personal use | Poem body |
| Birthday Thanksgiving | `BirthdayThanksgiving.ttf` | — | (kept for future) |
| Dawning of a New Day | `DawningofaNewDay.ttf` | SIL OFL | (kept for future) |
| Little Days | `LittleDays.ttf` | — | Diary input text |
| Megalin | `Megalin.otf` | — | Tagline + "Generate verse" button |

To swap one in elsewhere, change the relevant CSS variable in `src/styles.css`
(`--font-handwritten`, `--font-letter`, `--font-display`, etc.).
